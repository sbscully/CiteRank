README

This is my attempt at using Google pagerank to rank academics papers.

The directory 'citerank (without tests)' contains my first attempt, without tests.  Run the file 'run_citerank.rb' at the terminal to see the top ten ranked papers, using the method outlined in the following paper:

http://iopscience.iop.org/1742-5468/2007/06/P06010/

'pagerank (with tests)' contains my second attempt, with tests.  The algorithm used passes all tests but is very inefficient.  I did not have time to improve the algorithm or include the citerank modifications.
